# API

Будущий backend-контур, если вынесем API отдельно от Next.js app.

В MVP можно начать с Next.js API routes/server actions внутри `apps/web`, а эту папку оставить под будущий отдельный backend.

Основные зоны:
- Telegram init data validation.
- Studio/catalog API.
- Booking API.
- Mock payments.
- Notifications.
