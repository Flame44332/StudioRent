# Design References: Telegram Mini App для поиска студий

Дата сохранения: 2026-06-07

## Референс: BeatChain

BeatChain взят как основной референс для Telegram Mini App с темной мобильной эстетикой, каталогом, поиском, профилем и нижней навигацией.

## Что важно сохранить

- Темная тема: почти черный фон, мягкие темно-серые панели, белый основной текст, серый вторичный текст.
- Интерфейс внутри Telegram Mini App: верхняя панель Telegram, компактная ширина, контент рассчитан на мобильный экран.
- Поиск вверху главного экрана: большая rounded search-панель, рядом иконки быстрых действий.
- Большой промо-баннер/карусель под поиском.
- Нижняя навигация с 5 разделами: главная, каталог/список, добавление/создание, избранное, профиль.
- Карточки с мягкими скруглениями, плотной компоновкой и минимумом декоративного шума.
- Секции каталога с заголовком слева и действием "Смотреть все" справа.
- Профиль с переключателем ролей: покупатель/продавец. Для нашей идеи это можно адаптировать как "Клиент" / "Студия".
- Пустые состояния внутри карточек: спокойный текст без лишних иллюстраций.
- Модальное окно поверх затемненного/размытого контента для важных действий, например привязки email.

## Как адаптировать под приложение студий

- Главный экран:
  - Поиск студии по названию, району, метро или адресу.
  - Кнопки быстрых действий рядом с поиском: карта, уведомления, фильтры.
  - Баннеры: новые студии, скидки, свободные окна сегодня, подборки.
  - Секции: "Популярные студии", "Свободно сегодня", "Рядом с вами", "Топ по оборудованию".

- Карточка студии:
  - Фото студии.
  - Название, район/метро.
  - Цена от N руб/час.
  - Краткий список оборудования: микрофон, мониторы, звуковая карта, пульт.
  - Рейтинг/отзывы.
  - Быстрый статус: есть свободные окна / ближайшее окно.

- Экран студии:
  - Галерея фото.
  - Оборудование.
  - Цены и пакеты.
  - Календарь свободных окон.
  - Кнопка бронирования и оплаты.
  - Правила студии и адрес на карте.

- Профиль:
  - Переключатель "Клиент" / "Студия".
  - Для клиента: история бронирований, избранное, документы/чеки, платежные данные.
  - Для студии: расписание, заявки, цены, оборудование, фото, статистика, выплаты.

- Навигация:
  - Главная.
  - Карта.
  - Бронирования.
  - Избранное.
  - Профиль.

## Визуальное направление

- Общий тон: премиальный, музыкальный, ночной, технологичный.
- Не делать перегруженный маркетинговый лендинг. Первый экран должен быть рабочим приложением.
- Основной интерфейс должен быть плотным и быстрым: поиск, карта, фильтры, карточки, бронирование.
- Акценты можно делать синим/фиолетовым/неоновым, но не превращать весь интерфейс в один градиент.
