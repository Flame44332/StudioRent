# Design Brief

## Основной референс

BeatChain: темный интерфейс Telegram Mini App, крупные карточки, нижняя навигация, плотная структура, быстрый доступ к каталогу.

## Принципы

- Первый экран должен быть приложением, а не лендингом.
- Главный сценарий: найти студию и забронировать время.
- Карта и список должны переключаться быстро.
- Карточка студии должна показывать фото, цену, район/метро, оборудование и ближайшие свободные окна.
- Интерфейс должен ощущаться нативно для Telegram WebView.

## MVP-экраны для дизайна

- Home/catalog
- Map
- Studio details
- Booking flow
- Mock payment
- User profile
- Studio dashboard
