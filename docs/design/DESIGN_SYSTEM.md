# Design System Notes

## Направление

- Telegram Mini App.
- Темная тема.
- Плотный, мобильный, рабочий интерфейс.
- BeatChain как главный референс по атмосфере и паттернам.
- Первый экран должен ощущаться как продуктовый каталог, а не лендинг.
- Figma MVP file: https://www.figma.com/design/e3iwwHTb7T7rrsPJWggrQl

## Базовые экраны

- Catalog/search.
- Studio card/list.
- Studio details with address and map.
- Booking.
- Payment mock.
- Profile.
- Studio dashboard.

## UI-паттерны

- Нижняя навигация на 5 пунктов.
- Поиск сверху.
- Быстрые фильтры по метро, цене, оборудованию и времени.
- Секции с заголовком и действием справа.
- Карточки студий.
- Карта внутри экрана студии, а не как главный первый экран.
- Модальные окна для важных действий.
- Переключатель роли: Клиент / Студия.
