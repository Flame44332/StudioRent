## Что строим

`StudioRent` - Telegram Mini App для поиска и бронирования студий звукозаписи в Москве.

Ключевая ставка: не просто каталог, а быстрый Telegram-native сценарий "найти подходящую студию для записи трека -> понять цену, оборудование и адрес -> забронировать свободное окно".

## Принятые решения

- Основной формат: Telegram Mini App.
- Рабочее название: StudioRent.
- Первый фокус: студии звукозаписи для записи треков в Москве.
- Первый экран: каталог студий.
- Карта: внутри карточки/экрана студии.
- Бронь в MVP: моментальное подтверждение.
- Будущий режим: подтверждение брони студией.
- Первые студии: московские студии звукозаписи, заносим вручную/сидом.
- Yandex Maps: JavaScript API.
- Геокодер: желательно добавить.
- Платежи: пока mock-заглушка.
- Предварительный стек: Next.js + TypeScript + Postgres + Prisma.
- Дизайн-референс: BeatChain, темная плотная мобильная эстетика.
