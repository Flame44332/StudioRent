# Integrations

## Telegram

Статус: токен сохранен в `.env.local`.

Нужно:
- Создать/настроить mini app URL у бота.
- Настроить кнопку запуска Web App.
- На backend проверять Telegram init data.
- Не отдавать `TELEGRAM_BOT_TOKEN` во frontend.

## Yandex Maps

Статус: ключ сохранен в `.env.local`.

Для v0:
- Показывать карту Москвы.
- Рисовать метки студий из нашей базы.
- Открывать карточку студии по клику на метку.

Дополнительно после v0:
- Геокодер для перевода адреса студии в координаты.
- Ограничение ключа по доменам в кабинете Яндекса.

## Payments

Статус: реальные платежи не делаем.

Для v0:
- Mock checkout.
- Красивый экран "оплата прошла".
- Booking status меняется как при реальной оплате.

После v0:
- Выбор провайдера: ЮKassa, CloudPayments, Т-Банк, Robokassa.
- Поддержка webhook, возвратов, чеков, статусов платежей.
