# PRD: StudioRent

Дата: 2026-06-07

## Идея

`StudioRent` помогает артистам, вокалистам, рэп/поп-исполнителям, битмейкерам и продюсерам быстро находить студии звукозаписи в Москве для записи треков, смотреть оборудование, фото, цены, адрес и бронировать свободные окна прямо внутри Telegram.

Для студий продукт в будущем дает автоматизацию расписания, заявок, витрины, бронирований и оплат.

## Основные роли

- Клиент: ищет студию, сравнивает варианты, бронирует слот.
- Студия: размещает профиль, управляет окнами, ценами, оборудованием и заявками.
- Админ платформы: модерирует студии, решает спорные кейсы, следит за качеством каталога.

## MVP-цель

Проверить, есть ли спрос на Telegram-native каталог студий звукозаписи с быстрым сравнением и моментальным бронированием слота без ручной переписки.

## MVP-гипотеза

Если пользователь может за 2-4 минуты найти подходящую студию для записи трека, увидеть цену/оборудование/адрес и забронировать окно, то продукт снижает friction по сравнению с поиском через Telegram-чаты, сайты студий, карты и ручные сообщения.

## MVP-ограничения

- Москва.
- Первый фокус: студии звукозаписи для треков.
- Первый экран: каталог студий.
- Карта показывается внутри карточки/экрана студии вместе с адресом и подробностями.
- Бронь подтверждается моментально.
- Подтверждение студией добавляется в планы на будущее.
- Только web Telegram Mini App.
- Платежи не подключаются, используется mock-заглушка оплаты.
- Первые московские студии заводятся вручную/сидом.
- Календарь бронирования может быть упрощенным на первом этапе.
