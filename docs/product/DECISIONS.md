# Product Decisions

Дата: 2026-06-07

## Зафиксированные решения

- Рабочее название: `StudioRent`.
- География MVP: Москва.
- Первый фокус: студии звукозаписи для записи треков.
- Не включаем в первый фокус: репетиционные базы, подкаст-студии, танцевальные/фото/лофт-площадки.
- Первый экран: каталог студий.
- Карта: не главный первый экран, а часть карточки/экрана студии вместе с адресом и подробностями.
- Бронь в MVP: моментальное подтверждение.
- Будущий режим: подтверждение брони студией вручную или гибридный режим по настройке студии.
- Первые студии: московские студии звукозаписи, заносим вручную/сидом для MVP.
- Платежи в MVP: mock-заглушка без реальной платежной интеграции.
- Дизайн-референс: темная Telegram Mini App эстетика BeatChain.

## Следствие для MVP

MVP должен в первую очередь показать сценарий:

1. Пользователь открывает `StudioRent`.
2. Видит каталог студий звукозаписи в Москве.
3. Фильтрует по метро, цене, оборудованию и свободному времени.
4. Открывает карточку студии.
5. Смотрит фото, оборудование, цену, адрес и карту.
6. Выбирает слот.
7. Проходит красивую mock-оплату.
8. Получает моментально подтвержденную бронь.
