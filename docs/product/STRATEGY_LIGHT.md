# Light Strategy

## Стартовая позиция

Делаем marketplace для бронирования музыкальных студий в Москве внутри Telegram. Первый продуктовый фокус - не широкий сервис услуг, а быстрый поиск студии и бронирование конкретного временного окна.

## Почему Telegram Mini App

- У целевой аудитории Telegram уже является рабочей средой для коммуникации.
- Не нужен отдельный app install.
- Можно быстро перейти от контента/бота/канала к бронированию.
- Уведомления о брони можно отправлять в тот же Telegram.

## MVP-угол

Для клиента:
- меньше переписок;
- понятные цены и свободные окна;
- быстрое сравнение студий;
- карта и фильтры.

Для студии:
- витрина в Telegram;
- автоматизация заявок;
- меньше ручного согласования;
- понятный календарь занятости.

## Рекомендация по запуску

1. Сначала собрать качественный каталог из 10-20 студий вручную.
2. Сделать красивый пользовательский flow: поиск -> студия -> слот -> mock-оплата -> подтверждение.
3. Параллельно сделать простой кабинет студии, но не перегружать его CRM-функциями.
4. После прототипа проверить спрос на 5-10 реальных студиях и 20-30 потенциальных клиентах.

## Риски стратегии

- Студии могут не захотеть отдавать реальные слоты без доверия.
- Клиенты могут продолжать писать напрямую, если нет выгоды в брони через приложение.
- Реальная платежка и отмены создадут юридические и операционные вопросы.
- Нужно быстро понять, кто платит: клиент комиссию, студия комиссию или студия подписку.
