# Уточняющие вопросы

## Уже отвечено

- Рабочее название: `StudioRent`.
- Первый фокус: студии звукозаписи для записи треков.
- Бронь в MVP: моментальная.
- Будущий режим: подтверждение брони студией.
- Первый экран: каталог.
- Карта: внутри карточки/экрана студии.
- Первые студии: московские студии звукозаписи, заносим вручную/сидом.

## Продукт

1. Кто первый основной пользователь внутри звукозаписи: артисты/вокалисты, рэп-исполнители, битмейкеры или продюсеры?
2. Какие услуги показываем первыми: запись вокала, запись трека под ключ, сведение/мастеринг, аренда комнаты без инженера?
3. Какие условия отмены показываем в MVP?
4. Какие отмены допустимы: бесплатная отмена, штраф, перенос, ручное решение?
5. Нужны ли отзывы в MVP или достаточно рейтинга-заглушки?

## Бизнес

1. Модель монетизации: комиссия с брони, подписка для студии, платное размещение, продвижение или гибрид?

3. Планируем ли работать только с юрлицами/ИП или с частными студиями тоже?
4. Нужна ли модерация каждой студии перед публикацией?
