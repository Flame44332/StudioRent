# MVP

## Версия 0.1

Цель: кликабельный и технически рабочий прототип Telegram Mini App `StudioRent` с каталогом московских студий звукозаписи, карточкой студии, картой внутри карточки и mock-бронированием.

## Основные экраны

1. Каталог студий с поиском, быстрыми фильтрами и подборками.
2. Список студий звукозаписи.
3. Карточка студии.
4. Экран студии: фото, оборудование, цены, адрес, карта, расписание.
5. Выбор даты и времени.
6. Mock-оплата.
7. Моментальное подтверждение брони.
8. Профиль клиента.
9. Базовый кабинет/витрина студии, если успеваем без задержки MVP.

## Основные сущности

- User
- Studio
- StudioPhoto
- Equipment
- Room
- Service
- AvailabilitySlot
- Booking
- MockPayment

## Приоритеты

- P0: Telegram auth, каталог, список студий, карточка студии, адрес/карта внутри студии, бронь, mock-оплата, подтверждение.
- P1: фильтры по метро/району, цене, оборудованию, свободному времени.
- P2: кабинет студии, редактирование слотов, подтверждение брони студией.
- P3: отзывы, избранное, промокоды, реальные платежи.
