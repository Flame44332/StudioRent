# StudioRent

Telegram Mini App для поиска и бронирования студий звукозаписи в Москве.

## Рабочая структура

- `apps/web` - frontend Mini App.
- `apps/api` - backend/API, если выделяем отдельно.
- `docs/product` - продуктовые документы, решения, MVP, вопросы.
- `docs/research` - рынок, конкуренты, аудитория.
- `docs/design` - дизайн-система, референсы, UX.
- `docs/architecture` - стек, модули, интеграции.
- `docs/operations` - запуск, окружения, ключи без секретов.
- `assets/references` - визуальные референсы, включая текущие скриншоты BeatChain.

## Секреты

Локальные ключи лежат в `.env.local`. Файл исключен из git через `.gitignore`.
